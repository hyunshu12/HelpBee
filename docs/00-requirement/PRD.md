# HelpBee PRD (Product Requirements Document)

## 제품 개요

**HelpBee**: AI 기반 스마트 양봉 진단 SaaS 플랫폼

- **목표**: 한국 중형 양봉가(200~500통)의 바로아 응애 조기 진단
- **MVP 출시**: 2025년 6월
- **핵심 가치**: 벌통 이미지 → AI 분석 → 감염 진단 리포트

## 사용자 스토리

### 양봉가 (주요 사용자)
- 벌통 이미지를 스마트폰으로 촬영하여 업로드할 수 있다
- AI가 분석한 바로아 응애 감염 위험도를 즉시 확인할 수 있다
- 감염 위험도에 따른 맞춤 조치사항 가이드를 받을 수 있다
- 과거 분석 이력을 대시보드에서 조회할 수 있다

### 관리자
- 사용자 가입/구독 현황을 모니터링할 수 있다
- AI 분석 로그를 조회하고 정확도를 검토할 수 있다

## 기능 요구사항

### P0 (MVP 필수)
- [ ] 이메일/비밀번호 회원가입 및 로그인
- [ ] 벌통 등록 및 관리 (CRUD)
- [ ] 이미지 업로드 (S3)
- [ ] AI 분석 요청 및 결과 조회
- [ ] 분석 리포트 생성

### P1 (MVP 이후)
- [ ] 소셜 로그인 (카카오, 구글)
- [ ] 분석 결과 이메일 알림
- [ ] 구독 플랜 (무료/Pro)
- [ ] 관리자 대시보드

## 비기능 요구사항

- API 응답 시간: p99 < 500ms
- AI 분석 응답 시간: < 30초
- 가용성: 99.9%
- 데이터 보안: 이미지 암호화 저장
